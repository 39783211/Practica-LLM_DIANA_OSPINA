# Practica-LLM_DIANA_OSPINA
# Proyecto: AI Escalation Assistant B2B

Objetivo: leer correos/escalamientos comerciales, clasificarlos, calcular días abiertos, asignar semáforo y generar una recomendación para liberar tiempo de los consultores.
En el futuro se tendrá mayor cantidad de correos que llegarán a un correo de escalamientos@empresa.com y se socializará los datos a incluir para tener una mejor métrica y poder construir dashboard
